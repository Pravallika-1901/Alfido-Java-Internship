# Alfido Tech Java Developer Internship

## Overview

This repository contains the tasks completed during my **Java Developer Internship at Alfido Tech**. Throughout this internship, I strengthened my understanding of Java programming fundamentals and gained hands-on experience by implementing practical applications.

## Tasks Completed

### Task 1 – Java Basics & OOP Fundamentals
- Implemented Java programs using loops, conditions, and arrays.
- Applied Object-Oriented Programming concepts such as classes, objects, inheritance, encapsulation, and polymorphism.

### Task 2 – Console-Based Mini Project
- Developed a console-based Student Management System.
- Used Scanner for user input.
- Implemented conditional statements and loops.

### Task 3 – Collections & Exception Handling
- Worked with ArrayList, HashMap, and HashSet.
- Implemented try-catch blocks for exception handling.
- Created and used a custom exception.

### Task 4 – File Handling & Basic Git Usage
- Performed file reading and writing using FileReader and FileWriter.
- Implemented exception handling for file operations.
- Managed project files using Git and GitHub.
